Установка расширения вручную

Эта инструкция поможет вам установить расширение Max Get User Token в браузеры на базе Chromium без использования официального магазина.

Подготовка: Убедитесь, что у вас есть папка с файлами расширения (manifest.json, content.js и иконка).

Google Chrome и Яндекс Браузер

  1. Откройте браузер и перейдите по адресу: chrome://extensions/ или browser://extensions/.
  2. В правом верхнем углу включите переключатель «Режим разработчика».
  3. Нажмите на появившуюся кнопку «Загрузить распакованное расширение».
  4. Выберите папку с файлами расширения и нажмите «Открыть».

Microsoft Edge

  1. Перейдите по адресу: edge://extensions/.
  2. В меню слева включите функцию «Режим разработчика».
  3. Нажмите «Загрузить распакованное» в основной части окна.
  4. Выберите папку с расширением.

Opera

  1. Перейдите по адресу: opera://extensions.
  2. Включите «Режим разработчика» в правом верхнем углу.
  3. Нажмите кнопку «Загрузить распакованное расширение».
  4. Выберите папку с расширением.
Важное примечание:

Не удаляйте и не перемещайте папку с файлами расширения после установки. Браузер запускает код напрямую из этой папки. Если вы переместите её, расширение перестанет работать.

При каждом перезапуске браузер может предупреждать, что вы используете стороннее расширение — это нормально, просто закрывайте уведомление.
